
1. git init > Initialize a new Git repository

2. git status > Show the status of the working directory and staging area

3. git add <file> > Stage changes for commit
   git add . > Stage all changes in the current directory

4. git commit -m "message" > Commit staged changes with a message

5. git log > Show commit history

6. git remote add origin <url> > Add a remote repository

7. git push origin <branch> > Push changes to the remote repository on the specified branch
